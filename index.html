O que é HTML?

HTML é uma linguagem de marcação usada para a Web. Ela se utiliza de marcações de texto por meio de códigos que informam ao navegador se aquilo escrito é um texto, link , imagem ou qualquer outra coisa !

Tags e Atributos

Tags são códigos interpretados pelo navegador , que formam a estrutura do site. Atributos são utilizados para atribuir funcionalidades para as Tags. Alguns atributos são gerais e outros específicos para cada Tag.

Algumas tags básicas que aprendi (todas estão entre < >):

  • html = arquivo html (todo o código vai dentro dessa tag)
  • head = cabeçalho, importante para adicionar intruções que não ficam visíveis na página. (ex: Metadados)
  • body = corpo (o que aparece para o usuário)
  • title = título da página (noem da aba)
  • i = itálico
  • u = sublinhado
  • mark = marca texto
  • sub = na parte inferior do texto
  • sup = na parte superior do texto
  • strong = negrito
  • input = entrada de algo
  • img = imagem
  • dell = faz um risco na no texto
  • font = muda a cor , tamanho ou fonte do texto dependendo do atributo utilizado

Alguns Atributos que aprendi:

Genéricos:

  • id = identificador (ex: < strong id="titulo">Meu site < /strong>)
  • style = formatar texto (ex: < strong style="color: blue" > texto < /strong>)
  • class = padronizar em umaclasse (ex: < strong class="titulo-principal">Meu site< /strong>)

Específicos:

  • < input type = "text" /> = entrada de texto(usado p formularios)
  • < input type = "number" /> = entrada numérica
  • < input type = "color" /> = seletor de cor
  • < img src = "endereco da imagem" /> = puxa a imagem pelo indereco (pode ser da internet)
  • < img width = "150" src="endereço da imagem" /> = largura da imagem
